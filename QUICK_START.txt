================================================================================
                       WHAT TO EXPECT
================================================================================

When you run the demo:

1. Camera window opens showing live webcam feed
2. Green rectangle shows the detection region (right side)
3. Press 'b' and wait for "Background captured" message
4. Show your hand gesture in the green box
5. The system displays:
   - Which gesture was recognized
   - Confidence score (%)
   - Current status

Supported Gestures:
   E = Yes/OK gesture
   L = Love gesture  
   F = Peace gesture
   V = Victory gesture
   B = Fist gesture

================================================================================
                        TROUBLESHOOTING
================================================================================

Problem: Camera window not opening
Solution: 
  - Check webcam is connected
  - Try running check_system.py to diagnose
  - Restart the application

Problem: "Background not captured" message stays
Solution:
  - Press 'b' again
  - Make sure you have a camera feed
  - Wait 2 seconds after capturing

Problem: No gestures recognized
Solution:
  - Improve lighting conditions
  - Make sure background is captured first
  - Show gesture more clearly in the green box
  - Move slower and hold the pose steady

Problem: Confidence score too low
Solution:
  - Threshold is 70% confidence
  - Make sure hand is clearly visible
  - Better lighting helps recognition
  - Adjust hand position in detection region
